# AI Excuse Generator

## Description
An AI-powered excuse generator built using Python, Streamlit, and Google Gemini API. It generates realistic and human-like excuses based on user input scenarios and also provides voice output and fake chat proof.

## Features
- AI-generated excuses using Gemini API  
- Fake chat proof simulation  
- Text-to-speech output (voice excuse)  
- Emergency call simulation  

## Tech Stack
- Python  
- Streamlit  
- Google Generative AI (Gemini API)  
- gTTS  

## Project Structure
## Project Structure
app.py  
utils.py  
requirements.txt  

## How to Run

1. Install dependencies:
pip install -r requirements.txt

2. Set API key:

Windows:
set GEMINI_API_KEY=your_key

Mac/Linux:
export GEMINI_API_KEY=your_key

3. Run:
streamlit run app.py

## Example

Input: "I missed my class"  
Output: AI generates a realistic excuse with voice + chat proof
